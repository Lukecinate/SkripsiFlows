# SkripsiFlow

**Dari ide menjadi skripsi siap format.**

SkripsiFlow mengonversi Markdown, TXT, dan paste dari AI menjadi dokumen skripsi Indonesia yang terstruktur.

## Status

Fondasi aplikasi dan ingestion deterministik sudah tersedia. Parser, editor, template, citation, dan exporter dikembangkan bertahap.

## Menjalankan

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

Dokumentasi ada di `docs/`, riwayat perubahan di `doc-logs/`, dan konteks agent di `doc-nexts/`.
